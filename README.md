# LED Blink Project using Arduino

## Overview
This project demonstrates basic LED interfacing and GPIO control using an Arduino board. The LED connected to digital pin 13 blinks continuously with a custom timing pattern.

This project helps beginners understand:
- Arduino programming basics
- GPIO pin control
- Timing using delay functions
- Embedded systems fundamentals

---

## Components Required
- Arduino Uno (or compatible board)
- USB cable
- LED (optional)
- 220Ω resistor (for external LED)
- Breadboard and jumper wires (optional)

---

## Circuit Connection
- LED positive terminal → Pin 13
- LED negative terminal → GND

> Most Arduino boards already contain an onboard LED connected to pin 13.

---

## Working
The program continuously:
1. Turns the LED ON for 100 milliseconds
2. Turns the LED OFF for 900 milliseconds
3. Repeats the process infinitely
4. It has a duty cycle of 1 second.
